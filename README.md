Open the Tracking Map App.sln in Visual Studio Code 2022 with Maui and .NET packages installed already ( .NET Multiplatform App)


1) Install Android Emulator Google Pixel 7 APi 35 or use an Android device with USB Debugging Installed (Mine was Samsung Galaxy S23) 

2) dotnet restore  - will download required packages

3) run the application with the debug context and it will hotreload on the Android emulator
